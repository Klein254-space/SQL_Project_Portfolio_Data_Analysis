# Project_SQL

## Introduction

This project, **Project_SQL**, is part of my portfolio focused on practical SQL-driven data analysis. The repository demonstrates not only my proficiency in SQL but also my analytical thinking through real-world data exploration and insights.

### Background

In today's data-centric world, the ability to extract insights from raw data is invaluable. Through this project, I aimed to showcase how SQL can be leveraged to clean, analyze, and visualize complex datasets for actionable conclusions.

### Tools I Used

- **SQL** (Structured Query Language) for data querying, cleaning, transformation, and analysis.
- **PostgreSQL** as the database platform for storing, processing, and managing the data.
- **Visual Studio Code** as the primary code editor for writing and managing SQL scripts and documentation.
- **Git & GitHub** for version control, collaboration, and sharing the project with the wider community.

### The Analysis

The core of the analysis was implemented through the following queries, each targeting a specific data-related question:

1. **Top Paying Data Analyst Jobs**  
   This query identified the top 10 highest-paying remote Data Analyst job postings with specified salaries. Its purpose was to highlight lucrative opportunities in the remote job market for Data Analysts.  
   [View Query](https://github.com/Klein254-space/SQL_Project_Portfolio_Data_Analysis/blob/main/project_sql/1_top_paying_jobs.sql)

2. **Skills Required for Top Paying Data Analyst Jobs**  
   Building on the previous query, this analysis retrieved the specific skills required for the top 10 highest-paying Data Analyst roles. It helps job seekers understand which skills are sought after by employers offering the most competitive salaries.  
   [View Query](https://github.com/Klein254-space/SQL_Project_Portfolio_Data_Analysis/blob/main/project_sql/2_top_paying_job_skills.sql)

3. **Most Demanded Skills for Data Analyst Jobs**  
   This query determined the top 5 most in-demand skills for Data Analyst job postings, providing insights into which competencies are essential for success in this career path based on market demand.  
   [View Query](https://github.com/Klein254-space/SQL_Project_Portfolio_Data_Analysis/blob/main/project_sql/3_top_demanded_skills.sql)

4. **Top Paying Skills for Data Analysts**  
   Here, the focus was to determine which skills are associated with the highest average salaries for Data Analyst jobs, regardless of location. This assists professionals in prioritizing their skill development for financial gain.  
   [View Query](https://github.com/Klein254-space/SQL_Project_Portfolio_Data_Analysis/blob/main/project_sql/4_top_paying_skills.sql)

5. **Most Optimal Skills to Learn (High Demand and High Paying)**  
   This analysis combined demand and salary data to identify the skills that are both highly sought after and well-compensated, especially in remote roles. It guides career development by showcasing skills that offer both job security and lucrative pay.  
   [View Query](https://github.com/Klein254-space/SQL_Project_Portfolio_Data_Analysis/blob/main/project_sql/5_optimal_skills.sql)

Each SQL file is commented to explain the intention behind the query and the insights expected from its results.
